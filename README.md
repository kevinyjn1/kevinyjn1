# 🏆 Proud Project

## League of Legends Match Outcome Predictor
- **Description:** A data-driven model that predicts the outcome of a League of Legends match using 25-minute in-game metrics sourced from OraclesElixir.
- **Key Features:**
  - Baseline logistic regression model with >70% accuracy on test set
  - Exploratory Data Analysis with interactive Plotly dashboards
  - Clean data pipelines built with Pandas and NumPy
- **Tech Stack:** Python · pandas · NumPy · scikit-learn · Plotly
- **Live Demo:** [League of Legends Analysis](https://mrtuge123.github.io/League_of_Legends_Analysis/)
- **Source Code:** [View on Github]([https://github.com/kevinyjn1/League_of_Legends_Analysis](https://github.com/MrTuge123/League_of_Legends_Analysis))
