# üèÜ Proud Project: Checkpoint Champion ‚Äì Predicting Winners at 25

**Authors:**  
- Xingzhi Cui (tigercui@umich.edu)  
- Yun Jong Na (kevinyjn@umich.edu)  

**Description:**  
A classification pipeline that leverages in-game metrics at the 25-minute mark to predict League of Legends match outcomes. We process ~9,800 matches (two team-level records each) from OraclesElixir, clean and engineer features, then compare logistic regression variants to find the optimal sparse model.

**Key Highlights:**
- **Baseline Logistic Regression:** 81.25% test accuracy, ROC-AUC 0.8976  
- **Final LASSO Model:** 83.61% test accuracy, ROC-AUC 0.9214 with just 9 non-zero coefficients  
- **Top Predictors:** gold diff, experience, creep score, kills/deaths at minute 25, plus map side  
- **Multicollinearity Handling:** L1 regularization to stabilize estimates and auto-select features
